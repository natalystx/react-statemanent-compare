# Steps to use

## Install packages

first `npm install`
then `npm run install-all`

## start projects

`npm start`

## start an individual project

<b>Jotai </br></b>
`npm run jotai`

<b>React context </br></b>
`npm run react-context`

<b>Zustand </br></b>
`npm run zustand`

    Hope you happy with it.
